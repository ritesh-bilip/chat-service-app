Neon Chatroom: 

   a real-time chat application powered by Django Channels + WebSockets,       deployed seamlessly on Render with PostgreSQL and Redis for live messaging.

🔑 Key Highlights

       Real-time messaging with instant sync across rooms

       JWT authentication via my own Dasri Auth service

       Room-based chat – just enter a room name and connect

       Responsive glass morphism UI for a modern look

       Zero-downtime deployment on Render

⚙️ Tech Stack

       Backend: Django 5.1 + Channels 4.2 + Redis

      Database: PostgreSQL (Render free tier)

      Auth: JWT tokens from Dasri Auth

     Frontend: Vanilla HTML/CSS/JS WebSockets

    Deploy: Render



      Source code: https://github.com/ritesh-bilip?tab=repositories

👉 Enter room "test", use a JWT from my auth service, and experience live messaging instantly.
